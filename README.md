# WooCommerce SMS Auth Modal

یک افزونه احراز هویت مستقل و سراسری برای ووکامرس مبتنی بر رمز یکبار مصرف (OTP) پیامکی در قالب یک مودال پاپ‌آپ، بدون دستکاری فیلدهای پیش‌فرض تسویه حساب.

## توضیحات

این افزونه یک سیستم ورود و ثبت‌نام بدون رمز عبور (Passwordless) برای فروشگاه‌های ووکامرس فراهم می‌کند. کاربران می‌توانند تنها با وارد کردن شماره موبایل خود و تایید کد پیامکی، در سایت ثبت‌نام کرده یا وارد شوند. تمام عملیات در یک مودال زیبا و واکنش‌گرا انجام می‌شود که با Tailwind CSS ساخته شده است.

### ویژگی‌ها

- **مودال احراز هویت مستقل**: بدون نیاز به تغییر در صفحه تسویه حساب ووکامرس
- **ثبت‌نام و ورود بدون رمز عبور**: احراز هویت تنها با کد یکبار مصرف (OTP) پیامکی
- **پشتیبان از ۴ درگاه پیامکی ایرانی**: فراز اس‌ام‌اس، کاوه‌نگار، ملی‌پیامک، Sms.ir
- **معماری قابل توسعه**: استفاده از الگوی Factory برای افزودن درگاه‌های جدید
- **پنل مدیریت جامع**: تنظیمات ظاهری، مدیریت درگاه‌ها و تست پیامک
- **Rate Limiting**: محدودیت نرخ درخواست برای جلوگیری از سوءاستفاده
- **امنیت بالا**: تایید Nonce، پاکسازی ورودی‌ها، انقضای خودکار کد OTP
- **اتصال سفارشات مهمان**: اتصال خودکار سفارش‌های قبلی مهمان به حساب کاربر جدید
- **واکنش‌گرا و مدرن**: رابط کاربری با Tailwind CSS و Lucide Icons

### تکنولوژی‌ها

- WordPress Plugin API
- WooCommerce
- WP REST API
- Vanilla JavaScript (ES6+)
- Tailwind CSS
- Lucide Icons

### الزامات

- وردپرس ۵.۸ یا بالاتر
- ووکامرس ۶.۰ یا بالاتر
- PHP 7.4 یا بالاتر

### نصب

1. پوشه `wc-sms-auth-modal` را به مسیر `/wp-content/plugins/` کپی کنید
2. افزونه را از پنل ادمین وردپرس فعال کنید
3. به مسیر **ووکامرس > تنظیمات ورود پیامکی** بروید و تنظیمات را پیکربندی کنید

### ساختار

```
wc-sms-auth-modal/
├── wc-sms-auth-modal.php          # فایل اصلی افزونه
├── assets/
│   ├── css/                       # استایل‌های فرانت و ادمین
│   └── js/                        # اسکریپت‌های فرانت و ادمین
├── includes/
│   ├── admin/                     # پنل مدیریت
│   ├── api/                       # REST API
│   └── gateways/                  # درگاه‌های پیامکی (Factory Pattern)
└── languages/                     # فایل‌های ترجمه
```
